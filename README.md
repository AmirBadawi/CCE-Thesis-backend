# OpenAI Chatbot
it is a chatbot project

Instructions to run:
- Clone this repo
- CD inside
- create folder named: .venv
- open terminal and run: python -m venv .venv
- activate env: .venv/Scripts/activate
- run: pip install -r requirements.txt
- run: python app/main.py
